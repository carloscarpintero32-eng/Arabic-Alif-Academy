EXAMPLE WORD AUDIO FILES
=========================

This directory contains audio files for example words in each letter lesson.
Each letter has 4 example words (isolated, initial, medial, final).

File Naming Convention:
-----------------------
letter-{ID}-{POSITION}.wav

Where:
- {ID} = Letter ID number (1-28)
- {POSITION} = isolated, initial, medial, or final

Total Files: 28 letters × 4 positions = 112 files

File Specifications:
--------------------
- Format: WAV
- Sample Rate: 44100 Hz recommended
- Duration: 1-3 seconds per word
- Quality: Clear pronunciation
- Voice: Native Arabic speaker

Notes:
------
- Each file should pronounce the complete Arabic word
- Audio should be clear and suitable for language learning
- Missing files will show console warnings but won't break the app
- The app handles missing files gracefully during development
