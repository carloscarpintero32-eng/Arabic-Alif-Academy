SOUND FILES PLACEHOLDER
========================

This directory should contain the following 7 WAV files:

Required Files:
---------------
1. click.wav          - General button click sound
2. select.wav         - Selection/navigation sound
3. correct.wav        - Correct answer feedback
4. incorrect.wav      - Incorrect answer feedback
5. success.wav        - Task completion sound
6. navigation.wav     - Page/screen navigation
7. celebration.wav    - Game completion celebration

Instructions:
-------------
- Replace this file with actual .wav audio files
- See SOUNDS_SETUP.md in the project root for download links
- Free sound libraries: Pixabay, Freesound.org, Mixkit
- Recommended format: 44100 Hz, 16-bit, WAV
- Keep files under 100KB each for fast loading

Note: The app will work without these files, but sounds won't play.
You'll see warnings in the browser console if files are missing.
